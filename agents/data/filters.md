# Product Filters

This file contains filter options for products, including age ranges and other filtering criteria.

## Age Filter

Age ranges for filtering products:

- **0 to 12 months**
- **1 - 3 years**
- **3 - 5 years**
- **5 - 7 years**
- **8 - 12 years**
- **13 - 17 years**
- **18 - 24 years**
- **Adults**
