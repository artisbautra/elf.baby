# Shop Categories

This file contains all discovered shop categories organized in a hierarchical structure (up to 2 levels).

## Format
Categories are listed in the format:
- **Level 1 Category** > **Level 2 Category**

If a shop only has a single-level category, it's listed as:
- **Category Name**

## Categories

- **All Categories**
- **Categories**
- **Categories for Him**
- **Categories for her**
- **Category Name**
- **Level 1 Category** > **Level 2 Category**
